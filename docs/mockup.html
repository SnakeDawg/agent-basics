Primitives

Five things, working together

Skills run procedures. Agents orchestrate them. Shared context lives separately. Scope decides what gets injected. Everything stays in plain markdown.

Skill

A procedure that runs when triggered. Tool-agnostic markdown. Self-contained. Stays generic — never reads shared/ context directly.

.claude/skills/<name>/SKILL.md

Agent

Orchestrator. Loads context, delegates to skills, synthesizes output. Thin or fan-out — scope decides.

.claude/agents/<name>.md

Shared

Read-only organizational context — personas, archetypes, strategy, competitors. Loaded by agents only. Never published.

.claude/shared/...

Scope

Per-session manifest of what context applies. Activity slash commands resolve the chain and hand the filtered set to skills.

/scope <path>
