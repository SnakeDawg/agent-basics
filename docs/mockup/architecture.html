    Capability layers

    The four-layer model

    A semantic layering for any agentic system. Hover any layer to ignite its accent. Labels are placeholders — the layers stay; the names will adapt to your stack.

    Layer 01 · Intake

    UI / User intake

    Where the human enters intent. Slash commands, prompts, interview-driven flows.

    Layer 02 · Logic

    Skills / active logic

    Procedures that run. Tool-agnostic markdown. Stay generic; receive context from above.

    Layer 03 · Orchestration

    Agent backbone

    Agents resolve scope, load context, delegate to skills, synthesize output.

    Layer 04 · Safety

    Memory / safety

    Read-only context, publish gates, validators, audit trails. The guardrails.
