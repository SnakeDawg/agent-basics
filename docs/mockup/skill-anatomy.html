The conversation with leadership. "Market intelligence" is not a single prompt. It's a 9-phase workflow (~700 lines) with mandatory source archival, sub-task checkpointing for resumability, an autonomous strategic-gap-analysis pass before review, 5 hard design principles, 6 golden-source templates, 2 reference modules, edge-case handling for paywalls and quiet users and conflicting feedback, and a 90-second heartbeat rule for long-running phases. Plus citation discipline, draft versioning, and HTML rendering for delivery. Two downstream skills already consume the output. That's the rigor. The diagram is the artifact you show.
