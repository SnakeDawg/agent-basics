Skill anatomy

What's actually inside one skill

Example: market-intelligence. The compact SKILL.md at the center is what triggers and runs. References load on demand. Inputs from the orchestrating agent come in on the left; outputs go out the bottom.

The boss conversation. "Market intelligence" is not a single prompt — it's a structured procedure (~650 words, always loaded), three on-demand reference modules totalling ~5,000 words, a briefing-path branch, a type-classification step, conditional output sections, a gap-analysis pass before delivery, and citation discipline. Plus the orchestrator pre-filters inputs based on the active scope. The "rigor" is not in any one piece — it's in the whole.
